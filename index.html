🐱 CLIP: Cat Image Understanding Project

This interactive demo explores OpenAI's CLIP model for multimodal alignment using a small set of cat images and natural language prompts.

📦 Dataset

  • 10 cat images from the COCO dataset
  • 10 descriptive prompts written by the user

🧠 What We Did

  • Used CLIP to embed image and text into the same semantic space
  • Calculated cosine similarity between each image-prompt pair
  • Visualized the results as a matrix and probability diagram
